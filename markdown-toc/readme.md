# 用于生成markdown目录

## 匹配格式

以#开头，并且正文与#之间只存在一个空格

例如`# title`

## 使用方法

`py toc.py target.md`

这样会在终端打印最终结果，如果想输出到指定文件，则采用重定向

`py toc.py target.md > gen.md`

## 设计方法

该程序思路非常简单，打开文件后对每一行进行处理，判断是否是标题使用有限状态机设计，其正则表达式为`#+ .+\n`

实际上就是判断是否是连续的`#`开头，并且后接一个空格

匹配到标题就生成最终的目录格式，目录格式采用`- [title](#title)`表示，对于缩进，与`#`个数有关，在`-`前添加对`#`个数减一乘2个空格
